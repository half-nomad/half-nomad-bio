```css
.portfolio-grid {
    display: grid;
    grid-template-columns: repeat(2, 1fr);
    gap: var(--spacing-2xl);
}

@media (min-width: 768px) {
    .portfolio-item.featured {
        grid-row: span 2; /* Featured 프로젝트 2줄 차지 */
    }
}
```
